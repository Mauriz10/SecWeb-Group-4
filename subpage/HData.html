Handling of data

To get access to the data you need to authenticate, that you have the rights to handle data. You get entry to the database with your login data, which you should get from the it-support. Make sure that you don't give any information or data to people who don't have the rights to handle data or to stranger. If you will give some data to third parties, it will have legal consequences for you. If you print some important documents, make sure that the documents will be locked in a filing cabinet or that the files will be destroyed if they are not needed anymore.

This guide needs to be used for every kind of data and specially for following data:

  • Customer Data
  • Human Resource Data
  • Accounting Data
  • Sales Data
  • Delivering Data
