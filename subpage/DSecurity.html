Database Security

For the Database Security it is important to install a firewall with the right settings. Administration should only have access to the settings from the firewall. Also, it is important that the roles of every employee are defined clearly. That you can guarantee that only the administrators are allowed to edit or delete things form the database. Another important fact is, that you need to encrypt the data and specially the login data. This needs to be done to make sure that a person who does not have the permission get access to the database can't use the data.
